Gaze Tracking Canvas

Visual Target Management System

by Niall Brown - Early Childhood Vision Consultant (ECVC)

Important: Canvas Not Saved

This app is for quick and flexible manipulation of 2D visual targets (images, videos, shapes). It will not save the arrangements.

Export Information

This tool allows you to create and export complex visual scenes for 2D image assessment. Combine various visual targets, backgrounds, and foregrounds to create scenes for 2D image assessment.

Salient Feature Highlighter

The Salient Feature Highlighter helps children with CVI identify key features in images by tracing outlines with bright, high-contrast colors. Use the pencil tool to highlight important visual features, helping to direct attention and support visual learning.
